# Quick Quiz App 🎯

An interactive quiz application built using HTML, CSS, and JavaScript. The app allows users to test their general knowledge through multiple-choice questions across different difficulty levels while tracking scores, maintaining a leaderboard, and reviewing answers after completion.

## 🚀 Features

* Three difficulty levels:

  * Easy (5 Questions, 20 seconds each)
  * Medium (6 Questions, 15 seconds each)
  * Hard (7 Questions, 10 seconds each)

* Timer-based quiz system

* Dynamic score tracking

* Progress indicator and question counter

* Instant answer feedback

* Local leaderboard using browser Local Storage

* Detailed answer review screen

* Randomized question selection

* Fully responsive modern UI

* Dark-themed design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Browser Local Storage API

## ⚙️ How It Works

1. User selects a difficulty level.
2. User enters their name.
3. Questions are randomly selected and shuffled.
4. Each question has a time limit based on difficulty.
5. Correct answers increase the score.
6. Scores are saved in Local Storage.
7. Results are displayed after quiz completion.
8. Users can review all answers and attempt the quiz again.
